# l33t.ltd python api wrapper

## Description
Python 2/3 library for the [anti-captcha](https://l33t.ltd) web site

### Getting APi key
Register at [l33t website](https://l33t.ltd). Go to the main page and click Profile. Follow "Infromation" tab. Here you can see your API key
